# Швидкий старт - Аналіз чатів через Playwright

## 🚀 НАЙПРОСТІШИЙ СПОСІБ - 1 КЛІК:

### Подвійний клік на:
```
RUN.bat
```

**Це все!** Скрипт автоматично:
- ✅ Запустить Chrome з потрібними параметрами
- ✅ Відкриє чат ChatGPT
- ✅ Перевірить підключення
- ✅ Запустить аналіз

Вам потрібно лише:
1. Мати файл `@mt_offer.txt` з чатами (формат: ID\tНазва\t@username)
2. Переконатись що ви авторизовані в ChatGPT

**Ліміт:** За один запуск обробляється **10 чатів** (щоб уникнути FloodWait)

---

## 📝 Альтернатива - ручний запуск (3 кроки):

### 3️⃣ Запустіть скрипт

python analyze_chats_playwright.py

## 🎚️ Налаштування ліміту:

Якщо хочете обробляти більше/менше чатів за раз:

Відредагуйте [analyze_chats_playwright.py](analyze_chats_playwright.py:42):
```python
MAX_CHATS_PER_RUN = 10  # Змініть на потрібну кількість
```

**Рекомендації:**
- 10-20 чатів - безпечно для одного акаунту
- 50+ чатів - може спричинити FloodWait
- Затримка між чатами: 3 секунди (налаштовується)
