# Аналіз чатів через Playwright + ChatGPT

Скрипт для автоматичного аналізу Telegram чатів через ChatGPT з використанням Playwright.

**Переваги над Selenium:**
- ✅ Підключення до вже запущеного Chrome (не потрібна повторна авторизація)
- ✅ Робота в **ОДНОМУ конкретному чаті** ChatGPT (не створює нові вкладки)
- ✅ Стабільніше і швидше
- ✅ Краща обробка помилок

**Чат ChatGPT для роботи:**
https://chatgpt.com/c/6937098c-d498-832a-8921-8e543d15ff2f

**Використовується Chrome** - один браузер, просто і зручно!

## Встановлення

### 1. Встановіть залежності

```bash
pip install playwright
playwright install chromium
```

Або з requirements.txt:
```bash
pip install -r requirements.txt
playwright install chromium
```

## Налаштування

### Змінити порт Edge

Відредагуйте в скрипті `analyze_chats_playwright.py` (рядок 29):
```python
EDGE_CDP_PORT = 9222  # Змініть на інший порт
```

Та запустіть Edge з цим портом:
```batch
msedge.exe --remote-debugging-port=ВАШЕ_ЧИСЛО --user-data-dir="C:\edge-playwright-profile"
```

### Змінити затримки

В скрипті (рядки 32-33):
```python
TYPING_DELAY = 50  # мс між натисканнями клавіш
WAIT_FOR_RESPONSE = 20  # секунд очікування відповіді від GPT
```

### Змінити промпт для ChatGPT

Відредагуйте промпт у функції `generate_descriptions_with_chatgpt()` (рядок 324):

```python
base_prompt = """Ваш власний промпт тут..."""
```

## Troubleshooting

### ❌ "Не вдалося підключитись до Edge"

**Рішення:**
1. Переконайтесь що Edge запущено з параметрами
2. Перевірте що порт 9222 вільний
3. Спробуйте перезапустити Edge

**Перевірка підключення:**
Відкрийте в браузері: http://localhost:9222/json
Якщо бачите JSON з відкритими вкладками - все ОК

### ❌ "Не знайдено поле вводу ChatGPT"

**Причини:**
1. Ви не авторизовані в ChatGPT
2. ChatGPT змінив дизайн (треба оновити селектори)
3. ChatGPT показує помилку

**Рішення:**
1. Авторизуйтесь на https://chat.openai.com/
2. Переконайтесь що чат відкритий і працює
3. Спробуйте вручну відправити повідомлення

### ❌ "FloodWait від Telegram"

Це нормально - Telegram обмежує кількість запитів.
Скрипт автоматично чекає потрібний час.

### ⚠️ ChatGPT не відповідає

**Причини:**
1. ChatGPT перевантажений
2. Ліміти безкоштовного плану
3. Треба більше часу на відповідь

**Рішення:**
1. Збільшіть `WAIT_FOR_RESPONSE` в коді
2. Спробуйте пізніше
3. Використовуйте ChatGPT Plus

### 🔄 Браузер закривається після скрипта

Це НЕ повинно відбуватись!

Скрипт спеціально НЕ закриває браузер - він залишається відкритим для наступних запусків.

Якщо браузер закривається:
- Не запускайте Edge через звичайний ярлик паралельно
- Використовуйте тільки Edge з параметрами

## Порівняння: Selenium vs Playwright

| Характеристика | Selenium | Playwright |
|---|---|---|
| Підключення до існуючого браузера | Складно | Легко (CDP) |
| Авторизація | Кожен раз | Один раз |
| Швидкість | Повільніше | Швидше |
| Стабільність | Середня | Висока |
| Налаштування | Багато | Мінімум |

## Поради

1. **Тестуйте на 1-2 чатах спочатку** - переконайтесь що все працює
2. **Не обробляйте більше 10 чатів за раз** - ChatGPT може дати ліміт
3. **Тримайте Edge відкритим** - не треба перезапускати при кожному аналізі
4. **Перевіряйте результати** - іноді GPT може дати загальний опис
5. **Редагуйте промпт** - налаштуйте під свої потреби
