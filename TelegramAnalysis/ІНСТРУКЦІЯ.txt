  АВТОМАТИЧНИЙ АНАЛІЗ TELEGRAM ЧАТІВ
═══════════════════════════════════════════════════════════════════════════════

🚀 ШВИДКИЙ ЗАПУСК (найпростіший спосіб):

  Подвійний клік на файл: START.bat

  Скрипт автоматично:
  ✓ Запустить Chrome/Edge
  ✓ Відкриє ChatGPT
  ✓ Перевірить авторизацію
  ✓ Запустить аналіз усіх чатів
  ✓ Розподілить результати по файлам


📋 ЩО РОБИТЬ СКРИПТ:

  1. Збирає дані з Telegram (через Telethon)
     • Тільки чати з активністю ≥20 повідомлень/день
     • Збирає повідомлення за останні 7 днів

  2. Аналізує через ChatGPT (для кожного чату):
     • Визначає категорію (10 категорій + "Інше")
     • Перевіряє мову (потрібно 80%+ російська)
     • Перевіряє заборонений контент
     • Створює опис чату

  3. Розподіляє чати по власникам:
     • @Commyx_Founder: Фріланс, Маркетинг, Арбітраж, Маркетплейси, Інвестиції
     • @Commyx_Manager: Crypto, Дизайн, IT, Експати


📁 ДЕ ШУКАТИ РЕЗУЛЬТАТИ:

  output/detailed/
    ├─ founder_chats_detailed.txt  ← Детальний опис для Founder
    └─ manager_chats_detailed.txt  ← Детальний опис для Manager

  output/compact/
    ├─ founder_chats_compact.txt   ← Короткий формат для швидкої відправки
    └─ manager_chats_compact.txt   ← Короткий формат для швидкої відправки

  output/rejected/
    └─ rejected_chats.txt          ← Відхилені чати (з причинами)

  output/other/
    └─ other_category_chats.txt    ← Категорія "Інше"


⏱️ СКІЛЬКИ ЧАСУ:

  • Збір даних: 5-10 хвилин (150 чатів)
  • ChatGPT аналіз: 5-10 хвилин (~93 активні чати × 3-5 сек)
  • ВСЬОГО: ~15-20 хвилин


⚙️ НАЛАШТУВАННЯ:

  Файл: analyze_chats_playwright.py

  • MAX_CHATS_PER_RUN = 150      ← Скільки чатів обробити за раз
  • MIN_MESSAGES_PER_DAY = 20    ← Мінімальна активність
  • HIGH_ACTIVITY_THRESHOLD = 40 ← Поріг HIGH активності
  • CHATGPT_URL = "..."          ← URL чату ChatGPT


📊 РІВНІ АКТИВНОСТІ:

  • HIGH: ≥40 повідомлень/день
  • LOW:  20-39 повідомлень/день
  • Відхилено: <20 повідомлень/день


❗ ВАЖЛИВО:

  1. НЕ ЗАКРИВАЙТЕ браузер під час роботи!
  2. НЕ ПЕРЕКЛЮЧАЙТЕСЯ на інші вкладки ChatGPT
  3. Переконайтесь що є інтернет
  4. Перевірте що ви авторизовані в ChatGPT


🔧 РУЧНИЙ ЗАПУСК (якщо потрібно):

  python RUN_ALL.py


📞 ПІДТРИМКА:

  Якщо виникли проблеми:
  1. Перевірте що Chrome/Edge встановлено
  2. Перевірте що авторизовані в ChatGPT
  3. Перевірте що є інтернет
  4. Подивіться логи в консолі
